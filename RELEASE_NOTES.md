# iPhone Media Sync v1.4.0

Four new features — all fully local, no paid services.

## New in 1.4.0
- **Full-size preview** — double-click any thumbnail (Backup, Probably Delete,
  or Free up space) to view the full image, with its details.
- **Quarantine browser** — a new **Quarantine…** toolbar button lists the local
  copies made before deletion, so you can **restore** any of them, delete them
  permanently, or empty the folder. Deletion is now fully reversible.
- **Offline text detection (OCR)** in the Probably Delete tab — a **Refine with
  text detection** button runs the bundled Tesseract engine on the shown
  candidates to confirm which actually contain text (messages/memes) and drop
  likely false positives. Bundled into the installer; no separate download, no
  cost.
- **Overview tab** — a dashboard: totals (photos/videos/Live/screenshots),
  backed-up vs. not, a by-year breakdown, and your largest items.
